# Great Authors

Ten legendary author personas (Hemingway, McCarthy, Didion, Baldwin, McPhee, Wallace, Orwell, King, Le Guin, Vonnegut) plus workflow slash commands for prose craft and editorial work. Companion to [`great-minds-plugin`](https://github.com/sethshoultes/great-minds-plugin).

**Status:** Design phase. See [`docs/superpowers/specs/2026-04-24-great-authors-plugin-design.md`](docs/superpowers/specs/2026-04-24-great-authors-plugin-design.md) for the full spec.
